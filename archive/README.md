# Archive

This directory contains legacy, duplicated, generated, or superseded material that no longer belongs in the project root.

## Layout

- `docs-legacy/root/`: old root-level manuals, summaries, and setup notes kept only for historical reference
- `docs-legacy/guides/`: old guide-format binaries/documents superseded by the current Markdown manuals
- `generated/`: temporary/generated artifacts that should not live in the root

Nothing in this directory is required for the app to run.
